# Existing Code Context

## Already Implemented (Backend)
- **CacheService** (`src/main/services/cache.service.ts`):
  - `downloadTrack(track)` - Downloads single track
  - `cancelDownload(trackId)` - Cancels download
  - `deleteTrack(trackId)` - Deletes single cached track
  - `clearCache()` - Clears all cached tracks
  - `isCached(trackId)` - Checks if track is cached
  - `getCachedPath(trackId)` - Returns file path if cached
  - `getStats()` - Returns cache statistics
  - `getCachedTracks()` - Returns cached tracks (minimal data)

- **PlayerService** (`src/main/services/player.service.ts`):
  - `getStreamUrl(track)` - Returns `file://` URL if cached, otherwise stream URL
  - Offline mode check in `play()` - Blocks non-cached tracks in offline mode
  - Offline mode skip in `next()/previous()` - Skips non-cached tracks

- **Database** (`src/main/database/database.ts`):
  - `audio_cache` table with track_id, album_id, file_path, file_size, cached_at, last_accessed_at
  - Methods: getCacheEntry, addCacheEntry, deleteCacheEntry, getAllCacheEntries, getCacheTotalSize, clearCache

## Already Implemented (Frontend)
- **Store** (`src/renderer/store/store.ts`):
  - `CacheSlice` with downloadTrack, deleteFromCache, clearCache, fetchCacheStats, fetchCachedTrackIds
  - `cachedTrackIds: Set<string>` - Tracks currently cached
  - `cachedAlbumIds: Set<string>` - Albums fully cached
  - `downloadingTracks: Set<string>` - Tracks currently downloading

- **AlbumDetailView** (`src/renderer/components/Collection/AlbumDetailView.tsx`):
  - Shows cached indicator (green dot) next to cached tracks
  - Shows downloading indicator for in-progress downloads
  - Context menu has "Download" option for individual tracks

## IPC Channels Already Registered
- `cache:download-track` - Download single track
- `cache:delete-track` - Delete single track
- `cache:clear` - Clear entire cache
- `cache:get-stats` - Get cache statistics
- `cache:get-cached-tracks` - Get cached tracks list
- `cache:is-cached` - Check if track is cached

## What Needs to Be Added
1. **Album caching** - downloadAlbum method in CacheService + IPC + Store
2. **Album cache deletion** - deleteAlbum method in CacheService + IPC + Store
3. **Cache management UI** - New CacheView component to see all cached tracks
4. **Enhanced getCachedTracks** - Include full track details, not just minimal data
